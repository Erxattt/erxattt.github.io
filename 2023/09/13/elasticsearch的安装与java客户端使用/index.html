elasticsearch的安装与java客户端使用

什么是 elasticsearch

非常强大的开源搜索引擎

组成

kibana,logstash,beats -> ELK (elastic stack)

  • logstash beats 数据抓取
  • kibana 数据可视化
  • elasticsearch 存储,计算,搜索数据 基于 lucene java 搜索引擎类库（基于倒排索引）

概念

什么是文档和词条

  • 每一条数据就是一个文档
  • 对文档的内容分词，得到的词语就是词条
    什么是正向索引
  • 基于文档 id 创建索引,逐个去查是否包含字段

什么是倒排索引

  • 对文档内容分词，对词条创建索引，并记录词条所在的文档的信息，查询时先根据词条查询到文档 id，而后获取到文档
