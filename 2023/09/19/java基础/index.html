  • 编译阶段

编译阶段

xxx.java 叫做源文件 里面的代码叫做源代码
.java 要进行编译(编译要进行语法检查)后 生成 .class 文件叫做字节码文件 无法在操作系统中直接执行
javac是jdk中的工具 用来编译 java源文件
用法：dos执行 javac <java源文件路径>

运行阶段

主要通过java命令进行
如果有个编译完成的A.class文件，格式：java 类名 后面不能加路径
直接 java A 即可运行，运行此命令会启动java虚拟机即jvm，jvm会启动类加载器ClassLoader，类加载器会把A.class文件解释成操作系统可以理解的二进制可执行文件。
